# 🤖 Simple Python Chatbot

A simple rule-based chatbot built using Python.
This project demonstrates basic string matching and user interaction in the terminal.

## 🚀 Features

* Responds to greetings (hello, hi, hey)
* Handles farewells (bye, goodbye)
* Answers "how are you"
* Displays current time ⏰
* Simple and beginner-friendly

## 🛠️ Technologies Used

* Python 3
* datetime module

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/MAGNOS710/simple-chatbot.git
```

2. Navigate to the project folder:

```
cd simple-chatbot
```

3. Run the script:

```
python chatbot.py
```

## 💡 Example

```
YOU: hello
Chatbot: Hello! How can I assist you today?

YOU: what time is it
Chatbot: The current time is 14:32:10.
```
